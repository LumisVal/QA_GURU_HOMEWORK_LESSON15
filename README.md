# UI автотесты для DemoQA

## Описание проекта

Проект содержит UI-автотесты для сайта [demoqa.com](https://demoqa.com/) с использованием **Python + Pytest + Selene + Selenium + Allure**.

Объект тестирования — формы и базовые UI-компоненты сайта DemoQA.

Покрыты сценарии:
- заполнение формы регистрации студента;
- проверка открытия страницы формы;
- проверка отображения обязательных полей;
- выбор значения в radio button;
- заполнение формы Text Box;
- работа с Check Box;
- добавление и поиск записи в Web Tables.

* * *

## Используемый стек

Python • Pytest • Selene • Selenium • Allure Report • GitHub • Jenkins • Selenoid

## Что проверяют тесты

Тесты проверяют работу основных компонентов на сайте `demoqa.com`:

### Student Registration Form
- открытие страницы формы;
- отображение заголовка формы;
- отображение основных обязательных полей;
- выбор пола;
- отправка формы только с обязательными полями;
- успешная полная регистрация студента.

### Text Box
- заполнение полей:
  - `Full Name`
  - `Email`
  - `Current Address`
  - `Permanent Address`
- проверка корректного отображения введенных данных после отправки формы.

### Check Box
- раскрытие дерева элементов;
- выбор checkbox `Desktop`;
- проверка отображения результата выбора.

### Radio Button
- выбор значения `Yes`;
- выбор значения `Impressive`;
- проверка появления корректного текста результата.

### Web Tables
- добавление новой записи в таблицу;
- поиск записи по значению;
- проверка отображения найденной записи.
